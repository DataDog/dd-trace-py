# ===== Wrapper composite actions =====
# `.github/actions/<wrapper>/action.yml` files reference `DataDog/<action>@<sha>`
# upstream — that is exactly what the wrappers exist to centralize. They are
# NOT allowlisted with `:*` anymore; instead, `check-external-refs.py` applies
# the stricter `wrapper-uses-not-sha-pinned` pattern to wrapper files, which
# requires every `uses:` line to be SHA-pinned (40 hex chars). A tag or branch
# ref like `@v1.0.4` or `@main` slipped into a wrapper would fail lint.
